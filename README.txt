Novashop - Static demo site
Files:
- index.html (home)
- products.html (product listing)
- about.html
- contact.html (contact + whatsapp)
- cart.html (cart + order form)
- css/styles.css
- js/app.js
- products.json
- assets/* (svg placeholders)

To run locally:
Open index.html in a browser (no server required). For full behaviour, use a simple static server:
python -m http.server 8000
then open http://localhost:8000
